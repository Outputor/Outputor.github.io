既要仰望监察，又要脚踏实地

发表于 2018-10-07 | 分类于 随笔 | 浏览 次
字数统计: 6.6k | 阅读时长 ≈ 22

写在最前

转眼间，还有三个月，2018年就要过去了，真的是如白驹过隙，今天是国庆小长假的第6天，正好有时间把前九个月的数据整理一下，看看这九个月都干了个啥，我知道肯定是惨不忍睹，可是这个勇气还是要有的。

用数据说话

通过自己记录的客观的数据对我自己进行分析。有时候自我感觉良好其实并不然。

睡眠时间



平均每天睡眠不到7个小时，最长一次是7月29号的时候在济南和老婆约会，睡了10：35分，我是猪吗。。。最短的时候一天睡了3：39，在5月4号去通宵和老爸干活。众位数集中在6到7个小时。
对于这个睡眠时间我还是比较满意的，最起码我没有每天都软绵绵的，整天没有睡醒的样子。暑假的时候也没有多睡多少，一周也就46个小时左右。
综上，我不是那种喜欢睡懒觉的人。这个下季度继续保持
但是，睡的有点晚这个问题始终没有解决，现在晚上11：30睡觉就是非常早了的，一般都在到0：00左右了，如果我可以早睡半个小时，然后早起半 个小时的话，早上头脑清醒的时候可以多做点事（写日记，看书，背单词，做反思，学菜谱……），是非常理想的一件事，可是最近貌似就没有早睡过。

通勤时间



这个通勤时间如果有的星期花费的比较多的话，肯定是这个星期骑车去别的地方了，或者坐公交车去别的地方了。总是来说，这些数据是可以反映我每天骑车上下学的时间的，一天在家有三个来回，一次需要10多分钟，这就需要差一多一个来小时，因为还是来回学校通勤的时间比较多，所以就算是有些单个数据比较高也不会对我造成很大的影响。
通勤时间是不可避免的，我觉得他们在宿舍住的人应该花费的时间会比我更多吧，毕竟他们是走着来回的。
一天一个小时，或许是我最小的通勤时间了，等我有了工作单程一个小时还差不多。。。

身体健康


关于身体健康这块真的是说来惭愧，40周里有7周竟然时间为0，也说明对于身体我还是不够上心呀，要不然会掉头发这么严重。
我的身体健康，主要指的是运动，而我的运动，今年主要指的是跑步，因为健身房今年就没有去过，后期如果把泳池开了的话就可以去游泳了。以后天也冷了，不能再跑步了，正好去游泳。
等过了马拉松，就不去操场跑步了。
如果说身体健康，只谈运动，是不是有点狭隘呢，我能想到的身体健康是这个样子的

  • 先要说是防脱发的问题，这个是一个很严重的问题，现在看到公交车上那些中年卸顶了大叔，我就害怕，所以防脱发也是身体健康的一部分
  • 其次是体重的问题，我的体重已经严重不达标，之前没有注意到这个问题，现在在镜子里看到自己瘦的不行，还有些驼背，这个形象简直是大打折扣
  • 还有驼背的问题，现在我已经有点注意了，但是，还不够，这个应该可以通过健身来解决，珂皓的身板就很正，他比我还要高，一点也没有驼背
  • 我想得到一个比较有型的身材，不要练的全都是肌肉块，只要穿衣显瘦，脱衣有肉就行了
  • 最后是保持身体机能处于一个健康的状态，不要三天两头感冒，每天都可以痛快的拉屎，脸上不要天天爆痘，不要今天这里疼，那里不舒服的

    社交时间


社交时间比较不真实，因为当初设定这个选项的时候，我是没有明确的定义的，识简单地划分为朋友聊天和朋友小聚两个选项，每个里面也没有详细的内容。就是一个大而空的没有落地的项目，比如每周要给一个同学打电话，但是呢，最近也没有在做了。会在微信上给同学聊聊天，嘘寒问暖几句。
我一直信奉的是通过一来一回的交往加深友情。并且这种刻意的交往是会加深感情的。不是那种虚情假意的说。
我这个记录方法有问题，而不是统计的问题，也有可能是没有深刻理解这个问题
何以谓之为社交，当我们社交时，我们在干什么？聊天，吃饭，然后呢？？？没啦。。。
看罗胖的逻辑思维，讲到为什么要成为了个受欢迎的人，因为我们需要别人的帮助，我们永远不知道什么时候会需要什么人的帮助，可能是扫地的清洁工阿姨，也有可能是我们许久未联系的朋友。
不知为啥，突然挺想新疆的那帮朋友的。
说了一堆乱七八糟的话，还是没有找到重点。
归结起来，是对时间统计的时候没有太注意区分，忘记了这个社交的属性是向外的，需要有两个人共同完成，所以我记录的是有问题的。
关于社交时间的记录，以后的任务就是做到精准记录，因为这人是和外界进行交流才产生的一段时间，所以，我们非常有必要记录下来交互的对象是谁，怎么个样子，方便以后回顾的时候用，这个东西在时光似金里面改。

休闲娱乐

关于休闲娱乐就不放表格了，只给出一个数据参考一下，那就是一共在这个上面花费了393个小时，然后我有216个小时是在手机上度过的。也就是说玩手机花了216个小时，这是一个很恐怖的数据，也就是说我每天平均花费47min在手机上干无聊的事情（逛淘宝，刷微信，看qq等一些无聊的事情）。
关于休闲娱乐，我的初衷就是把用它来监测自己到底玩了多长时间的手机，但是如果翻看自己的记录，耍手机用的时间非常的多，但是耍手机干了个啥，是没有记录，这个其实也是无效记录，因为不知道自己到底耍手机干了个啥，以后怎么再对手机进行节制呀。这是非常让人蛋疼的事。还是那件事，记录不明。以后应该精确和尽量多的记录，就怕说不明白在干啥。
还有一个问题就是休闲娱乐的分类做的不好，有好多都是那些只有一个记录的临时的项，这就导致在透视表里看的非常的不清楚。
重新分类，只有分清楚了才能更好的做统计。
靠，这个excel筛选之后做填充简直是个bug呀经过对于分类的思考，前两级目录不能变化，第三级菜单必须是较少变化的，然后，后面用逗号来分隔说明的时候一定要详细，尽量做到都有四级目录
ok，改版之后的时光似金更加的合理了。简直是大爱。
这次大改的是社交，休闲以及家庭生活。
改版本的原则是把之前的三级目录改成四级目录了。记录更加的详细，并且给每一个三级目录在后面强加上了逗号，这样就可以强迫自己对内容进行输入了。

学习成长


这个数据很让人不开心呀。总的用来学习成长自我提升的时间非常少，只有557h，平均每周只有14h20min，远低于我当初设想的25h/w，时间像乳沟一样，挤挤总会有的。
主要从以下几个方面来展开
阅读、思考、英语、数据人生、技能Get、Outputor、条理

阅读



上半年前九个月阅读总时间只有126个小时，如果看柱状图的的话，这离散性，我靠，简直了，数据的方差肯定大的不行，有的周只有不到一个小时的时间，有的周却有12个小时，并且还有7周没有阅读。。。这说明我的阅读没有持续性，就是有些时候心血来潮了，干上一周，有的时候这一周可能比较忙，然后就懒的不想看了。真是蛋疼。。。
果然，读书不能只靠激情呀
并且还有一个bug，就是不知道我这九个月里看了几本书，哪些书看完了，哪些书没有看完，每本书看了几遍，这个在记录方法上进行改进。

思考



我是从第11周开始做反思的，也就是一共做了有29周的反思，用时94：45，平均每周3：15，平均每天只有27min的时间在反思自己。我自己当初设想的是1h，这与目标有点远呀。
根据统计结果一周反思的时间达标时间大约在4个小时，也就说每天有半个小时的时间来做反思，从图上中以看出来，从刚开始做反思的是12-25周做的还是比较好的（从第11周开始把反思这件事纳入我的体系之中），到了第25周之后，就再也没有上过4个小时了。说明了热度衰减效应的存在，也就是说做任何一件事，都存在三分钟热度。不光做反思，读书也是。
应该怎么来弄这个事呢？？？不清楚
不知道如何来应对这件事，那么就思考对这件事，怎么做是肯定错误的。只要把这些错误避免了，不就可以把这件事做好了吗?
关于3分钟热度，这个问题一定要好好考虑一下要如何解决。

英语


在英语单词学习这个层面上，热度衰减效应就表现的更厉害了。当时刚用Anki，感觉很新鲜，热情持续了五周左右，然后呢，后面就再也没有上过6个小时了。一直到后面有两个星期压根就没有背单词。
年初的时候给自己定下了词汇过万的目标，然后搁置到现在，哈利波特里面的生词都没有背完，简直是惭愧到无地自容。荒废了一个假期，针对这种一停下来就不想再干，并且会一持续好长时间的行为怎么制止呢，也是一个头疼的问题

数据人生


关于整理数据并且复盘这个事，我自己是知道这是多么重要的一件事，并且还有一个问题就是今年日程表用的很不理想。Google日历现在也被我荒废了。
唉，一复盘，怎么日子过得这么水。都干了个啥子。。。

技能Get

这个软件硬件学习怎么现在感觉是一个比较鸡肋的项目呢，本来是想自己在学有余力的时候学习一些软件，比如PS，SU，Python，Excel等等，但是呢，又会与一些其它的内容相重合，比如学习Ansys和SAP2000这种软件，并不是单纯地为了学习软件而学习的，一般是和科研的工作放在一起的，也就是说在科研的时候学习软件，并不是单纯地为了知道这个软件怎么用，有什么功能而去学习。这就不能放在这里了。
那么这个项目到底要不要保留呢，那就这样吧，不把它叫做软硬件学习了，太难听了，叫技能GET吧
这样的话，我可以把各种各样的技能都算进去，比如用了10分钟学会了如何给word编不同的页码，都可以计算进去。而硬件上面比如练习开车、修电脑等等

Outputor


首先，完成篇文章的时间肯定要长于这个平均用时的，因为有些时间没有统计在内，跑到别的地方去了。
其次，完成文章的数量太少了。当年说好的一周一篇呢。
输出的素材还是积累的不够多。有些时候，有了好的思路，没有把它记下来，就没了。。。

条理

生活条理，定期整理资料这件事是荒芜了。有的时候有周花了好长时间去整理照片，去整理硬盘，有时候一连好几周没动静，就让电脑桌面上的东西塞的满满的。。。。这就是懒癌病的症状吧。

工作事业

研究生活


所谓的工作事业，对于我而言，就是研究生期间的学习，第九周开学到现在，已经干了30周了。时间可是真够长的，可是还没有一个可以拿得出手的东西来。没有论文，没人专利，也没有取得什么突破性的进展，我的课题也刚在上周确定。狗子，时间都去哪儿了。。。
总共花了1385个小时在我的研究生学习上面，比我在学习成长（557h）可多的不是一点半点。
算下来，花了429个小时在这个比赛上面（2018.7.13-2018.10.04），为了它，我变成邋遢模样，为了它，我脱了头发……
按照我之前的愿景，就是把每天工作8小时的时间全部用在工作事业上，也就是全部拿来搞科研，然后把其它时间，比如早上起来的空余的时间和晚上的时间拿来自我提升上。两条腿走路嘛，不过现在看来，有点跛，再这样下去就得瘸了。这尼玛可不行。

家庭生活


因为要和父母一起住，所以花在家庭和生活上的时间会相对多一些，如果只算回家做饭吃饭的话，一天4-5个小时为就够了，一周30个小时左右；偶尔可以陪父母家人去玩一玩，参加一下家庭的聚会，一周最多到40个小时吧；如果时间上了50的话，肯定是这一周家里有些别的事，不光是回家做饭吃饭这么单调了，比如老姐生小孩，去帮老爸的忙……
截止到十月一日，本年度与老婆共同花费的时间有314h，平均每周8h，平均每天1个小时多一点，当然这些时间里包含约会的时间，约会其实花费了很多时间的，因为什么都不干，就跟老婆腻在一起了。
在吃饭和做饭上面有647h，平均每周16.5h，每天2.3h，相较于吃食堂来说，肯定是多的不少，因为吃食堂的话也一顿饭也就20min。

总结：

  • 以后做时间记录的时候一定要把随时记录记在心里，就是当我们进行任务切换的时候一定要注意在时光似金上点一下，就ok了。现在我数据还是有点粗，用起来不很友好，就是找不到我具体的时间去哪儿了
  • 做事情都存在3分钟热度，对于这个现象，后面要做一下好好的总结与反思，看看问题出在什么地方，其实之前已经总结过了，但是之前总结的内容也存在三分钟效应，搁现在也不好使了
  • 通过这些总结，对时光似金的内容又做了一次调整，这次调整主要针对的是一些与别人沟通的项目。应该是更加具有合理性了。希望可以通过这一次的调整，让自己以后记录的更加方便，内容更加完善，更加符合自己的预期
  • 日程表要再次启用，其实日程表还是很有用的，这次日程表主要集中于自我学习与成长上面，而对于大把的工作时间不做过多的约束，之前就是因为对这个把握不好，总想把一周的时间塞的满满的，然后自己每周都完不成，没有成就感，头就炸了，就逐渐的放弃了
  • 下一阶段的读书重点在得到里面的内容整理上，边听精英日课边整理资料，把里面的东西变成自己的，再辅以Anki，用以记住一些知识点
  • 不要让自己的时间管理变成一潭死水，要每周都复盘，让它活起来，这种日子已经很久没有过了。自从暑假到现在，就没有好好审视过自己的时间。
  • 与人共同花费的时间是没有琐事这一项的，因为琐事是属于自己的事。

    再来看第二个长期目标，也就是入伙一家公司，这里面其实包含着两层含义，一个是先找到自己想要入伙的公司，第二是个自己可以达到公司收留的标准，也就是有所谓的投名状。再围绕一个更加长期的目标，就是五到十年后的安身立命的问题，找到一个明星公司，可以长期发展的职业，这个问题已经很急迫了，因为明年这个时候正是我找工作的时候，我可不想去施工或者设计院。。。只有一年的时间了。
    达到自己想要去的公司的留下自己，这个竞争必然是非常激烈的，按部就班地搞科研是不可能了。只有一个路子，就是提升自己解决问题的能力，提高自己思考问题的能力，在遇到问题的时候把解决问题的过程记录下来，写在简历里，在求职的时候用口头把它说出来。基于此，我要多读书，我思考，多储备一些将来有用的知识。这也是我在第一个目标之外要做的事情，也就是所谓的8个小时工作之外的时间你都在干啥决定了以后你能走多远。
    如果第一个目标是每天上班打卡干，而第二个目标则是为跳槽做的准备工作。
    第三个目标比较简单，还有不到两的时间，提前预祝自己新婚快乐。

    任务

    以终为始，想想自己最终想要得到什么。
    想得到的是一个规律的生活与学习状态，每天可以很充实而不虚度
    首先是要坚持用时光似金进行精准记录，周天做每周复盘，并且不要花太多的时间在这个上面
    然后配合日程表，制定每周的日程和事项，日程里只需要安排自己每天除了上班之外要干的工作，也就是八个小时之外的事情，然后就是下周上班可能要做的事列个表，或者写入日历中去也可以，但是不要安排具体的时间。因为这个时间一般是不可控的
    工作的任务主要有两个，一个是用三个月的时间把一个论文投出去，另一个是写一个专利，写那个节点的专利，看看之前有没有人写，利用其它时间搞学术，学软件，做模拟。
    工作之外的任务就是把提升自己的时间了。
    首先，每日反思是必须的，我愈加感受到每日反思的魅力之所在；
    其次，每日的阅读量必须要保证，初步想把万维钢老师的精英日课的内容用以阅读+记笔记的方法过一遍；
    然后，英语的学习也不能落下，主要有两个方面，一个是单词的积累，怎么也得对得起自己单词过万的梦想呀，因为单词背着背着就没有了，我需要再阅读一本新的英文书，初订RayDalio的《原则》，已经把中英文对照版本下载下来了。就这么愉快的决定了；
    用Anki来记住除了单词之外的东西，昨天突发奇想想把车标与所属的集团和国家用Anki记下来，也可以用来记住自己看到的书里的知识点,记住菜谱；
    除此之外，每周写一篇文章发到博客上那就可以用完美来形容了，可以把自己整理的读书笔记，工作中遇到的问题以及解决办法，自己反思的内容都发到上面去，这样压力就会小很多。
    学有余力的情况下学习更多的技能，想把Excel的VBA打通，不知道我的时间能不能安排下，还想学PS，SU，Vray,还想学Python，其实理性告诉我，我是不可能这么屌的。。。
    最后，关于生活方面，就是努力提升我的厨艺。
    打个比方，以上的目标就像是国家的战略层面，只是指出了一个方向；而刚才写的任务更像是省里接到了通知制定了一系列的方案；具体的制定计划，安排日程则更像是下面各级单位实施政策一要的做法，但是一般这时候下放到具体的人民身上这个可能就和国家的政策不一个味道了。学习也是同样的道理，制定了正确的方向与战略，最后落实到具体的做法上和一些细节的操作上总是与总的目标相差甚远，真是让人头秃。
    在制定日程的时候虽然小心又小心，可是总是有些日程是完成不了的，总是有些计划是赶不上变化的。这种情况是我一直在往里面跳的坑。
希望与志同道合的朋友共同交流与进步!!!
