鸡汤不鸡汤

发表于 2018-03-28 | 分类于 读书笔记
字数统计: 1.5k | 阅读时长 ≈ 4

《解忧杂货店》思维导图

这本书讲了什么？

故事发生在2012年伦敦奥运会之后，三个不良少年偷了东西之后，不料逃跑的车子却坏了，只得在半夜时分躲进了一个废弃的杂货店里。
这个杂货店就是解忧杂货店，30多年前的今天有一个叫浪矢的老人通过一个牛奶箱在这个店里给那些有各种各样的忧愁的人答疑解惑，虽然大部分都是小孩子的无聊恶搞，甚至有些还是一些脑筋急转弯的回答，可是老人却乐此不疲，以此为活下去的动力。
小说采用分叙手法，最后把所有的故事巧妙的连接在一起，最初感觉摸不着东西南北，看到最后才有一种恍然大明白的感觉。不由地发叹一声，“哦，原来如此呀。”
这里一共有5个主要咨询事件，分别是

  • 静美子咨询奥运会与照顾男友的问题
  • 克郎要不要坚持音乐梦想的事
  • 绿河(化名)咨询要不要将一个意外的孩子生下来的事
  • 浩介咨询要不要与父母一起出逃的事
  • 睛美咨询怎样挣更多的钱的事
    以上这五件事都有一个共通的地方，就是都与丸光幼儿园有关，里面的人有各种各样的关系，最后让我们得以窥见作者的用心。
    其中一和二是讲述的关于梦想的事情，套路大体相似，就是坚持梦想，不忘初心。
    其余三事咨询的关于人生转折的大事。一个母亲，一个孩子，一个迷茫的少女。
    最后他们的咨询有没有按照信中的指示去做，但是最后都是对老人以感激之情的，就连当然恶搞老爷爷不想上学还想考高分的小学生都对老爷爷心存感激。
    东野圭吾，一个悬疑小说家，就算写起小清新来也是让人欲罢不能。不得不让人佩服。

    作者想通过这本书表达什么？

    熟悉东野的人都知道，东野圭吾不是一个守旧的人，他不像有些作家一样，一直用一种自己熟悉的套路来写作，他是喜欢尝试新鲜写作方法的作家。可以毫不客气的说，在悬疑小说里，这是最温柔的；在鸡汤小说里，这是最刺激的。
    在我看来，作者这是对自己的突破，对新式的写作方式的探索。是作者积极主动探索的体现，这是东野圭吾的作品一直作为畅销书出现的原因之一。
    通过这一系列的小故事，作者想表达的不光是对梦想的追求，还有对爱情，亲情，友情，以及献身的精神的歌颂。

    这本书最让我喜欢的是它的什么？

    首先，是严密的逻辑以及恰到好处的伏笔。一篇经得起推敲的作品才是好作品，这部作品可以看出作者是精心设计了的，里面的一些东西需要我们自己好好领悟，比如在浪矢老爷爷让他儿子33年后发布杂货店要复活的消息的时候我是觉得有些突兀的，感觉这个情节安排的有些突兀，可是如果想到了连接未来的时候是老爷子不在杂货店的日子的话，就是说老爷子不在杂货店的日子里的信都是寄到了未来，未来都寄到了现在，可以收到那些信就得以解释了。还有在时间的安排上也没有出现混乱，这也是经得起推敲的，其实这本书就是讲了在33年前很短时间和33年后一个晚上的事。
    其次就是它的艺术气息，吹口琴的克郎，喜欢甲壳虫的浩介，以及作者对于这些细致的心理独白，有意识流的感觉，最后浩介终于知道了真相，又看到了《Let it be》的时候，让浩介仿佛终于明白了世间的一切。

    这本书与我有什么关系

    小说本来就是一种精神上的享受，不像一些具有实践指导意义的干货书籍，我觉得读罢全文，有一种莫名的酸爽就行了。最喜欢文章的第二个故事，一个为了梦想苦苦追寻的年轻人，在家庭与自己的梦想之间徘徊，因为能引起内心深处的共鸣所以心有感触，人总是感性又理性的动物，每个人扪心自问都有那似曾相识的最初的梦想，迷茫是每个年轻人身上特有的气质，对未来，对人生，对社会，处处充满着迷茫，我们要做的，可能并不是坚持自己最初的梦想那么简单的事了。因为，那毕竟是成功之后的谈资，并不属于我们。
    每个人在咨询之前内心都已经有了答案，只是在这里得到确认。这是小说里的主旨，让别人来帮我们做决定本身就是不负责任的体现，顺从你内心真正的想法，做自己最坚强的决定，即使最后没有得到自己想要的结果，也不会对自己做的事有半点后悔。
    总而言之，做自己的思考，做自己决定，做自己的主人。
    由于笔者实在是文笔拙劣，实在是写不出很好的美文，仅供自娱自乐。
