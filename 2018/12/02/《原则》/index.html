《原则》

发表于 2018-12-02 | 分类于 读书笔记 | 浏览 次
字数统计: 1.9k | 阅读时长 ≈ 6

做一个有原则的人

《原则》一书是华尔街一家传奇对冲基金公司桥水公司创始人瑞·达里奥关于生活的工作的原则的总结。达里奥是一个依靠原则行事的人。生活和工作中的所有问题的处理都是依靠原则来做决定的。作者本人及它的公司所取得的巨大成就就是依靠这一套行之有效的原则达到的。
本书主要分为三部分,分别是作者不同时间段的历程、作者的生活原则、作者的工作原则。而对于我最重要的部分是生活原则。从工作原则里我可以大体推断一下自己到底想找一个什么样的工作。
生活的原则主要分为以下几个方面
－　拥抱现实、应对现实
－　用五步流程实现人生愿望
－　做到头脑极度开放
－　理解人与人大不同
－　学习如何有效决策　
这些原则看起来更像是作者的人生感悟，作者奉行进化的原则，认为进化是宇宙中最强大的力量，是唯一永恒的东西，是一切的驱动力。而我们也是处于一个进化的过程之中的。进化可以看作是一系列的反馈环。有的人通过反馈取得进步，而有的人退步，有的人会原地不动。
达里奥相信痛苦的力量，没有痛苦就没有收获，认为痛苦是我们的必经之路，是取得巨大进步不可缺少的一步。
达里奥坚信头脑极度的开放，极度的透明，从更加理性的角度看待自己，而不是让杏仁核占据我们的头脑。头脑极度的开放不仅仅是虚心听取别人的建议，与别人分享自己的观点那么被动的开放，而是主动寻求开放与透明。即强制性地让别人对自己进行评价，从更多的可信的人那里得到关于自己的评价，从而得到对自己更加客观的认知。这是我一直奉行的“行有不得，反求诸己”的进化版本。达里奥认为自己是很难看清自己的，并且过高地估计自己，这是每个人的天性。其实现实确实是这样的。每个人都认为自己很牛逼。而谦逊是后天学习来的。
达里奥关于实现自己的人生愿望有五步流程，首先有明确的目标，然后找到问题，并且不容忍问题，接下来诊断问题，找到问题的根源，对问题进行方案规划，最后坚定地从头到尾执行方案。读到这里。感觉更像鸡汤了。不就是简单的发现问题，解决问题吗？？？就我自己而言，草草的读完这一遍，是不会对这五步流程有什么深入的了解的。因为这五步流程根本就没有实践，再怎样的纸上谈兵都是没有意义的。作者关于每一个流程，都做了详细的细部原则的划分。
刚开始不明白为什么作者要单独列一个大的标题来写如何有效决策，后来想了一想才明白过来，原来我们在处理问题的时候，就是做一个接一个决策的时候。因此决策的质量决定了最终做事的质量。更别提那些关键的决策了。因此做决策需要遵循一些重要的原则。而做决策最大的威胁就是我们的有害的情绪，也就是杏仁核宝宝。而一个好的决策是要分两步的，即先对要决策的事情做充分的了解然后再做决定。做决定的时候不是先假设一个决定再去寻找那个决定的证据。要多向这一方面可信度高的人请教。要分析变化的趋势而不必去死抠那些细节。其实决策的流程就是分析问题，解决问题的流程。并且要明白好的决策并不代表好的结果。
关于工作的原则我个人认为它更多地偏向于一个管理者应该持怎样的原则来管理公司，它与生活中的原则是相对应的。毫不违心的说，我喜欢这样的公司，我希望在这样的公司里上班。通过创意择优，让公司始终处于进化的下一秒。公司内部是一种极度求真、信任、透明的关系。我要做的是寻找这样的一家中国公司，然后加入它。
通过短短的14个小时的时间把《原则》这本书过了一遍，显然是很难深入理解和把握其中的精髓的。因此本书可以作为一个枕边读物，时常翻阅对照一下自己做事与思考问题的方法有没有遵循这些原则的地方。
其实达里奥所说的原则和《穷查理宝典》中查理所说的普世智慧是相近的，就是所谓的底层思维。达里奥在书里也反复提到，一个人的思维是很难做出改变的。因此他们的员工才会有一个长达18个月的适应期，并且会有很高的淘汰率，因此此时管理者认为这个人不适合公司的文化。而世面上关于成长的书所传达的一个基本的事实就是你要改变你的思维来适应现实的需要。其实，经过以上的推理，可是断定，我们读再多的书而不去做出改变也是没有意义的。不光是要做行动的实干者，更要做思想上的实干家。
作者在书里也说了，寻找最适合自己的原则并且把它养成一种习惯才是最重要的。其实就是这个样子的，如果只是照搬达里奥的原则而不加以思考和在真实的场景中运用，那只会像东施效颦一样，不会有任何进步。
接下来的任务是把达里奥的原则进行提炼，找到其中的80%，并且把这些原则放到自己的真实世界中去思考。在什么样的场景应该运用什么样的原则，在以前的什么场景中我还运用了这些原则。
首要尝试的是做到自己的极度开放与极度透明，找人去给自己提错误，从别人的视角去审视自己。就像给自己的游泳录像一样。要勇于回看录像，勇于回顾自己的错误与弱点。这些缺点正是我改进的地方。在痛苦中前进是一成不变的进化法则。
本书最大的收获以及接下来要改进的地方

  1. 保持头脑的极度开放与透明
  2. 痛苦是我们进步的基石，独立思考是形成自己原则的必要条件
  3. 通过与可信度高的人调换意见与互相评价从而认识到自己的弱点并加以改进
  4. 寻找并加入一家通过创意择优行事的公司
  5. 不断反思自己，形成自己的原则，改进自己的思想
希望与志同道合的朋友共同交流与进步!!!
