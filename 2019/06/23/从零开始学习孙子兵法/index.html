从零开始学习孙子兵法

发表于 2019-06-23 | 分类于 随笔 | 浏览 次
字数统计: 1.1k | 阅读时长 ≈ 3

这是一本可以研究一辈子的书

前言


孙子生于世界人类思想最富创造力的“轴心时代”。《孙子兵法》成书于公元前515年至512年，是中国的兵家经典，对后世的兵书产生了深刻影响；并且对世界各国的军事理论也产生了深远影响。《孙子兵法》一书所阐述的谋略思想和哲学思想，不仅用于军事上，对于政治、经济、国家、企业、 个人都有深远的影响。

《孙子兵法》共计13篇，共计6075个字。我读的是华杉老师的《华杉讲透<孙子兵法>》，华杉老师对十一家注的《孙子兵法》都有研究，然后集合这些内容，力图整理还原出最正本清源的《孙子兵法》

读了之后我感触颇深，我也不研究战争，也不研究军事，但是此书对于个人的成长也有很大的启发。虽然只有短短的6000多个字，但是6万字也不能把它里面包含的深刻思想讲清楚。我想开此博客对此书的每一篇采用正文+译文的方式进行学习。先知道它讲了啥意思，然后再加上所读书里面的案例，进行展开说明，最后再说一下对于自己的成长有什么帮助。毕竟，学习本书的目的是为了提升自己的认知。

始计第一

【原文】
孙子曰：兵者，国之大事也，死生之地，存亡之道，不可不察也

故经之以五事，校之以七计，而索其情。
一曰道，二曰天，三曰地，四曰将，五曰法。道者，令民于上同意，故可与之死，可与之生，而不畏危也。天者，阴阳、寒暑、时制也。地者，高下、远近、险易、广狭、死生也。将者，智、信、仁、勇、严也。法者，曲制、官道、主用也。凡此五者，将莫不闻，知之者胜，不知者不胜。
故校之以七计，而索其情，曰：主孰有道？将孰有能？天地孰得？法令孰行？兵众孰强？士卒孰练？赏罚孰明？吾以此知胜负矣。

将听吾计，用之必胜，留之；将不听吾计，用之必败，去之。计利以听，乃为之势，以佐其外。势者，因利而制权也。

兵者，诡道也。故能而示之不能，用而示之不用，近而示之远，远而示之近。利而诱之，乱而取之，实而备之，强而避之，怒而挠之，卑而骄之，佚而劳之，亲而离之，攻其无备，出其不意。此兵家之胜，不可先传也。

夫未战而庙算胜者，得算多也；未战而庙算不胜者，得算少也。多算胜，少算不胜，而况于无算乎！吾以此观之，胜负见矣。

【译文】
始计第一的计是指计算，敌我的力量，然后再看要不要打仗

孙子说：战争是一个国家的大事，关系到国家的生死存亡，不可以不仔细考察。

因此要从五个方面进行分析，对七种情况进行比较，得到详情来预测战争的胜负。
五个方面分别是道、天、地、将、法。道是指人民与君主一心，上下一心，共同生死，不畏危险。天是指昼夜、阴晴、寒暑、四季更替。地是指地势高下、远近、险易、宽广和狭隘、死地还是生地。将，指将领足智多谋、赏罚有信、关爱部下、勇敢果断、军纪严明。法，是指组织结构、权责划分、人员编制、管理制度、物资调配。对这五个方面，将领不能不深刻了解，对此了解的越多，就越容易取得胜利，不了解的将领肯定不能取得胜利。
所以要通过双方七种情况进行分析，得到其精确的情报并加以比较。哪一方的君主更加人道？哪一方的将领能力更强？哪一方占据天时地利？哪一方法令更加严格？哪一方士兵训练有素？哪一方赏罚分明？通过这些比较，我就知道哪一方可以取得胜利了。
